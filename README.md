## Dependências

### Ionic - CLI
- Instalação 
```
  npm install -g @ionic/cli
```
- Inicializando o ionic
```
  ionic start
```
- Hospedando localmente;
```
  ionic serve
```

### Adonis - CLI
- Iniciando o backend
```
  node ace migration:run
```

### NodeJS - versão 14 ou posterior.
- Para verificar a versão instalada, rode:
```
   node -v
```
